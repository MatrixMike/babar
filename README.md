# babar

The beginnings of a humble language


## Data Types
### Integers
### Decimals
```clojure
1 ;=> 1
2.3 ;=> 2.3
-3.4 ;=> 3.4
```

### Vectors
You can construct vectors by simply entering numbers with spaces
```clojure
1 2 3 4 ;=> [1 2 3 4]
```

## Operations

```clojure
+ 1 2 3 4 5 ;=> 15
```
